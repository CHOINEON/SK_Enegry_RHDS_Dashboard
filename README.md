## Description 
본 프로젝트는 S사의 No. 2 RHDS Fractionator, Separator 최적화를 위한 예측 모델의 AI 운전점 가이던스, 및 Explainable AI (XAI) 기능을 제공하는 모니터링 시스템을 리팩토링 중인 ver.2 입니다. 
이 시스템은 공정 데이터 기반의 예측 모델 결과와 경제성 분석을 통합하여 운전 효율을 극대화하도록 설계되었습니다.

 
## Overview
- 예측 모델: 주요 제품의 품질 예측 결과를 실시간 데이터와 함께 그래프로 시각화합니다.

- AI 운전점 가이던스 (HHPS / RHDS): AI 기반으로 최적 운전점(예: 투입구 제품 온도 등)을 제안하고, 이에 따른 예상 경제성 변화를 표출합니다.

- Explainable AI (XAI): 모델 예측 결과의 차이가 발생하는 원인을 입력 변수별 영향도 (Positive/Negative Relationship, Impact Comparison)를 통해 시각적으로 설명하여 운전자가 예측을 이해하고 신뢰할 수 있도록 돕습니다.
